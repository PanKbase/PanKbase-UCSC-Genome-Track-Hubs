Pancreatic Cell Type ATAC-seq Data

Description

This track hub contains ATAC-seq (Assay for Transposase-Accessible Chromatin using sequencing) data from various pancreatic cell types. The data appears to be from the Human Pancreas Analysis Program (HPAP).

Cell Types Included

  • A_Stellate: Activated stellate cells
  • Acinar: Acinar cells (enzyme-producing)
  • Alpha: Alpha cells (glucagon-producing)
  • Beta: Beta cells (insulin-producing)
  • Delta: Delta cells (somatostatin-producing)
  • Ductal: Ductal cells
  • Endothelial: Endothelial cells
  • Gamma: Gamma cells (pancreatic polypeptide-producing)
  • Immune: Immune cells
  • MUC5B_Ductal: MUC5B-positive ductal cells
  • Q_Stellate: Quiescent stellate cells
  • Schwann: Schwann cells

Track Types

For each cell type, there are two tracks:

  • Signal tracks (BigWig): Show the continuous ATAC-seq signal intensity
  • Peak tracks (BED): Show discrete regions of open chromatin (peaks)

Data Source

Data is hosted on AWS S3 and appears to be part of the Human Pancreas Analysis Program (HPAP) consortium.

Usage

Use these tracks to explore chromatin accessibility patterns across different pancreatic cell types. Signal tracks show the intensity of accessibility, while peak tracks highlight specific regulatory regions.
