Explore chromatin accessibility patterns across different pancreatic cell types

  • Reference map of islet cell type-specific chromatin accessibility from 97,659 cells derived from single nucleus ATAC-seq assays of 41 non-diabetic;T1D autoantibody positive (Aab+);T1D;and T2D donors from the Human Pancreas Analysis Program (HPAP)
  • Includes 12 distinct pancreatic cell types
  • Signal/coverage tracks (BigWig) - continuous ATAC-seq signal intensity
  • Peak tracks (BED) - discrete regions of open chromatin (peaks)
  • Resource URL: https://data.pankbase.org/analysis-sets/PKBDS0470WCHR/

Cell types by tracks:

  • A_Stellate: Activated stellate cells
  • Acinar: Acinar cells (enzyme-producing)
  • Alpha: Alpha cells (glucagon-producing)
  • Beta: Beta cells (insulin-producing)
  • Delta: Delta cells (somatostatin-producing)
  • Ductal: Ductal cells
  • Endothelial: Endothelial cells
  • Gamma: Gamma cells (pancreatic polypeptide-producing)
  • Immune: Immune cells
  • MUC5B_Ductal: MUC5B-positive ductal cells
  • Q_Stellate: Quiescent stellate cells
  • Schwann: Schwann cells
